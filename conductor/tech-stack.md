# 技术栈 (Tech Stack)

## 1. 核心语言 (Core Languages)
- **Python (>=3.10)**：用于后端逻辑、CLI 工具和数据库交互。
- **TypeScript (~5.9.3)**：用于前端开发，提供强类型支持以减少运行时错误。

## 2. 后端 (Backend)
- **FastAPI**：现代、高性能的 Web 框架，用于构建 RESTful API。
- **SQLAlchemy (>=2.0.0)**：功能强大的 SQL 工具包和对象关系映射器 (ORM)。
- **Typer**：基于 Python 类型提示的 CLI 构建库，用于实现管理员批量导入等工具。
- **Pydantic**：数据验证和设置管理，确保 API 输入/输出的结构化。

## 3. 前端 (Frontend)
- **Vue 3**：渐进式 JavaScript 框架，用于构建用户界面。
- **Element Plus**：基于 Vue 3 的组件库，提供丰富的 UI 组件库。
- **Vite**：下一代前端构建工具，提供极速的开发和构建体验。
- **Vue-i18n**：国际化插件，用于管理简体中文本地化字符串。

## 4. 开发与测试 (Dev & Test)
- **Pytest**：用于后端单元测试和集成测试，遵循 TDD 开发模式。
- **Vitest**：用于前端组件和逻辑测试。
- **Ruff**：极速的 Python 代码检查工具。
- **Vue-tsc**：针对 Vue 组件的 TypeScript 类型检查。

## 5. 基础设施 (Infrastructure)
- **Docker**：用于容器化部署，包含 Dockerfile 和 Docker Compose 配置。
- **Nginx**：作为前端资源的生产环境 Web 服务器。
