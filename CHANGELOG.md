# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-02-27

### Added
- **Auth (Frontend)**: 增强移动端及微信环境下的自动登录重定向。
- **UI (Frontend)**: 在仪表盘欢迎语中显示当前登录用户的姓名。

## [0.1.1] - 2026-02-27

### Changed
- **UI (Frontend)**: Improved visual feedback for selected assets (vehicles and gas cards) in the dashboard by changing the background color to `lightgreen` and locking it during hover states, especially for mobile devices.
